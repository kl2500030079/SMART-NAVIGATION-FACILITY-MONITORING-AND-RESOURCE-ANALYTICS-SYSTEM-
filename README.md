# Smart Navigation Facility Monitoring and Resource Analytics System

## Description
This project is a Python-based system designed to manage navigation, monitor facilities, and analyze resource usage within a campus.

## Features
- Add and view facilities
- Book facilities
- Complaint management system
- Asset management
- Navigation using BFS algorithm
- Data analysis using pandas
- Graph visualization using matplotlib

## Technologies Used
- Python
- JSON (for data storage)
- Pandas
- Matplotlib

## How to Run
1. Open the project folder
2. Run the Python file:
   python project.py

## Files Included
- python project.py (main code)
- python lab project.docx (documentation)
